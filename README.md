# LMLP 开发日志

这个 README 只记录 `dev-newFeature` 分支的开发过程和每个小版本做了什么。完整的项目介绍、安装说明、功能说明和截图展示放在 `main` 分支维护。

当前正式版：`v1.3.0`

当前构建：`1.3.14+mc1.20.6`

适配目标：Minecraft `1.20.6` / Fabric / Litematica / MaLiLib / REI

## 版本说明

### v1.3.14

- 将当前测试实例配置里的最小配方停止拆分列表写入默认配置：铁锭、金锭、粘液球、石英、蜂蜜瓶和红石。

### v1.3.13

- 交换数量显示样式1和样式2的顺序，默认样式1改为仅显示“盒数 + 组数 + 个数”。
- 原本带总数等号的显示方式调整为样式2。

### v1.3.12

- 样式3数量较长时，材料列表会按最长计数字符串自动扩展宽度，避免总计、缺少和可用列互相重叠。
- 数量显示样式配置的悬浮说明改用真正的乘号 `×`，并把样式3说明更新为 `A × SB + B × 64 (16) + C`。

### v1.3.11

- 新增数量显示样式配置，可在样式1到样式4之间切换材料数量的换算显示方式。
- 样式1只显示拆分结果，样式2显示“总数 = 盒数 + 组数 + 个数”，样式3显示 `A x SB + B x 64 + C`，样式4显示 Litematica 原版总数。

### v1.3.10

- 配方置顶重排动画扩展到所有已展开的子材料配方列表，不再只处理详情页顶层配方。
- 点击任意子材料里的置顶按钮后，该层配方框会按当前展开动画速度滑动到新的排序位置。

### v1.3.9

- 配方详情页置顶后，顶层配方框会按现有展开动画速度从原位置滑动到新的排序位置。
- 置顶重排时保持当前滚动位置，不再把页面跟随被置顶配方带到列表顶部，方便继续浏览剩余配方。

### v1.3.8

- 配方置顶按钮改用四态像素图标：已置顶、悬浮已置顶、未置顶、悬浮未置顶。
- 悬浮置顶按钮时会切换到对应悬浮图标，并显示白色边框和“置顶”提示。
- 清理上一版拆分星星资源，统一使用单张 `gui_widgets.png` 精灵表。

### v1.3.7

- 配方置顶按钮改用 32x32 暗金像素星星图标，保留无按钮底的原版像素风显示。
- 移除旧置顶图标的第三方素材说明和相关引用，项目资源重新回到自有素材。

### v1.3.6

- 将原木和木头混合候选项的替代材料名称统一显示为“任意原木”，例如橡木原木、橡木、去皮橡木原木、去皮橡木这一类组合。
- 沙子和红沙的候选项不再显示“任一”，直接显示为“沙子 / 红沙”。

### v1.3.5

- 配方置顶按钮改用实心/空心星星图标，不再绘制额外按钮底。

### v1.3.4

- 配方置顶按钮改为使用 Minecraft 配方书原版按钮底，并用像素风星标显示未置顶、悬停和已置顶状态。

### v1.3.3

- 新增配方置顶功能，可在配方详情页把某个配方设置为该输出物品的首选配方。
- 首选配方会保存到配置文件，并同步影响详情页排序、摘要页快速查看和递归材料拆分时使用的配方。
- 再次点击已置顶配方可以取消置顶。

### v1.3.2

- 摘要页和详情页的展开箭头现在会跟随展开/收起进度旋转，不再瞬间切换方向。
- 箭头旋转复用已有展开动画进度，保持和行高展开、裁切动画同步。

### v1.3.1

- 修复材料列表靠近底部的材料展开后，摘要面板底部仍需要额外向下滚动才能看全的问题。
- 展开动画期间滚动条会按展开完成后的目标高度预留空间，自动滚动也改为按目标材料底部精确计算。

### v1.3.0

- 将 `1.2.25+mc1.20.6` 开发构建整理为正式版发布。
- 完整梳理从容器界面打开材料列表、配方详情页、REI 转移按钮、悬停浮窗模式、展开动画等功能。
- GitHub Release 使用正式版 jar：`litematica-material-list-plus-1.3.0+mc1.20.6.jar`。

### v1.1.0

- 新增套娃式材料展开，材料可以继续拆解子材料，例如粘性活塞继续拆成活塞和粘液球，活塞还能继续往下展开。
- 摘要页支持内联树形展开，展开子材料时保留同级材料显示。
- 配方详情页支持对子材料继续展开，并直接渲染完整 REI 原生配方框。
- 详情页会显示 REI 返回的多个配方来源，改善白色染料等多来源材料的查看体验。
- 切石机、高炉等非工作台配方保留 REI 风格工作站小图标。
- 替代材料标签更短，例如“任意木板”，并循环显示可替代图标。
- 展开箭头改用 Minecraft 原版 `minecraft:recipe_book/page_forward` 资源，支持资源包适配。
- 材料悬浮信息改为右上角固定小面板，减少鼠标附近 tooltip 遮挡。
- 优化展开底部材料时的滚动范围，减少列表最底部内容看不全的问题。
- 新增 Minecraft `1.20.6` / Java `21` / REI `15.0.787` 适配。

### v1.0.1

- 修复配方详情页右上角“返回（Esc）”按钮在部分分辨率下与顶部物品信息框错位或重叠的问题。
- 修复顶部物品信息框中“总计 / 缺少”文本过长时超出黑色边框的问题。
- 当空间不足时，数量信息会自动拆成两行显示，并限制文本绘制范围。
- 新增 Minecraft `1.20.6` / REI `15` 的 tooltip API 适配。
- 发布 `1.20.1`、`1.20.4`、`1.20.6` 三个 Minecraft 版本对应 jar。

### v1.0.0

- 首个正式发布版本，面向 Minecraft `1.20.1` / `1.20.4`。
- 优化 Litematica 材料列表数量显示，支持“盒 / 组 / 个”格式。
- 单击材料行可展开配方摘要。
- `Shift + 单击`材料行可打开完整配方详情页。
- 配方详情页复用 REI 原生配方显示，支持物品 tooltip、高亮和交互。
- 支持熔炉、切石机等非工作台配方，并显示对应加工台图标。
- 保留 Litematica 原有材料统计、排序和导出行为。
