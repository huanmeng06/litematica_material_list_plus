# Litematica Material List Plus 1.0.0 for Minecraft 1.20.4

这是面向 Minecraft `1.20.4` 的适配版本。

## 下载文件

请下载并安装：

```text
litematica-material-list-plus-1.0.0+mc1.20.4.jar
```

## 主要功能

- 在 Litematica 材料列表中显示更易读的数量格式，例如 `9 盒 + 16 组 + 13 个`。
- 单击材料行可展开配方摘要。
- `Shift + 单击`材料行可打开配方详情页。
- 配方详情页复用 REI 的原生配方显示、物品悬停提示和物品交互。
- 对加工类配方显示对应工作站图标，并支持像 REI 物品槽一样交互。

## 安装环境

- Minecraft `1.20.4`
- Fabric Loader `0.19.2`
- Java `17`
- Fabric API `0.97.3+1.20.4`
- Litematica `0.17.4`
- MaLiLib `0.18.4-alpha.1`
- Roughly Enough Items / REI `14.1.786`

REI 是必要前置。Architectury API、Cloth Config API 和 Cloth Basic Math 等为 REI 相关前置，请按实际 REI 安装要求一并安装。

## 已知问题

- 目前仅对上方列出的 Minecraft、Litematica、MaLiLib、REI 与 Fabric API 版本组合进行适配与测试。
- 其他 Minecraft 版本或前置模组版本正在进一步适配中，暂不保证兼容。
