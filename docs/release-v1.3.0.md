> 这次更新把 Litematica Material List Plus 从“材料拆分增强”推进到更完整的材料准备工作流：可以从加工界面进入材料列表、查看 REI 风格的完整配方详情、递归展开材料树，并用更自然的动画理解层级关系。

## 新功能

| 功能 | 说明 |
| --- | --- |
| 容器界面打开材料列表 | 在工作台、高炉、切石机等容器界面中，可以使用 Litematica 原版材料列表热键打开材料列表，并保留原加工界面上下文。 |
| 配方详情页 | `Shift + 单击`材料行可打开完整配方详情页，按配方逐项展示材料来源。 |
| REI 原生配方显示 | 配方详情页尽量复用 REI 原生配方布局、物品交互、tooltip 和配方转移按钮。 |
| 配方转移按钮 | 从加工界面进入详情页时，可使用 REI 转移逻辑尝试把材料摆入当前加工台。 |
| 详情页热键返回 | 配方详情页可以使用材料列表热键返回原加工界面，再次打开时会恢复到上次详情页。 |
| 悬停浮窗三档模式 | `悬停浮窗模式` 支持 LMLP、Litematica 原版、不显示悬停浮窗三种选择。 |
| 展开/收起动画 | 配方详情页和材料列表摘要页的嵌套材料展开/收起加入短动画，层级变化更自然。 |

## 修复与体验优化

| 项目 | 说明 |
| --- | --- |
| 容器热键交互 | 优化从容器界面打开、返回、恢复详情页和搜索框输入的热键行为，减少与 MaLiLib 原生输入冲突。 |
| 详情页不暂停游戏 | 配方详情页不再暂停单人游戏世界。 |
| 详情页布局 | 优化标题、材料概览框和配方详情框的左侧对齐与垂直间距。 |
| REI tooltip 稳定性 | 修复 REI 转移按钮 tooltip、未知 tooltip 组件、收藏提示过滤和配方 ID 显示相关问题。 |
| 展开动画边框 | 修复配方详情页动画中内容框边框横线可能缺失的问题。 |
| 摘要页深层裁切 | 修复材料列表摘要页递归子材料动画中第二层及更深层级可能被错误裁切的问题。 |
| 多语言 | 补充中文、英文、香港繁体中文语言文件，并保持语言 key 对齐。 |

## 安装

1. 下载 Assets 中的 `litematica-material-list-plus-1.3.0+mc1.20.6.jar`。
2. 放入 Minecraft `1.20.6` Fabric 实例的 `mods` 文件夹。
3. 需要同时安装 Fabric API、Litematica、MaLiLib 和 REI。

适配目标：Minecraft `1.20.6` / Fabric / Litematica / MaLiLib / REI。
