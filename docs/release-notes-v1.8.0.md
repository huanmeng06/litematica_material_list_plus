# LMLP v1.8.0

v1.8.0 完成了从 REI 到 JEI 的全面迁移，并集中改进了配方交互、大型整合包性能、最小子材料计算和投影原点标记。所有受支持版本保持一致的操作方式与主要界面效果。

## ⚠️ 重要版本说明

> **LMLP v1.8.0 已全面迁移到 JEI，不再使用旧版 REI 集成。**
>
> 由于 JEI 对 Minecraft `1.21.4` 和 `1.21.5` 的适配情况无法满足本版本的依赖要求，v1.8.0 不再支持这两个 Minecraft 版本，也不会提供对应 JAR。它们的最后可用版本为 LMLP `v1.7.0`，请勿混用其他 Minecraft 版本的 v1.8.0 JAR。

本次 Release 提供以下 Minecraft Fabric 版本：

- `1.20.1`
- `1.20.4`
- `1.20.6`
- `1.21.1`
- `1.21.10`
- `1.21.11`
- `26.1.2`

## ✨ 新功能

| 功能 | 说明 |
| --- | --- |
| 全面迁移到 JEI | 配方解析、原生配方界面、配方与用途跳转、配方转移、快速合成和加工台入口全部改用 JEI；材料列表、配方摘要页和详情页的相关交互均已迁移。 |
| 配方浏览与快速合成 | 新增加工台连接板、配方转移状态按钮、可点击材料名称、摘要定位和自动展开；支持玩家背包 2×2 合成区，并保留工作台等 3×3 容器操作。 |
| 新版本支持 | 新增 Minecraft `1.21.10`、`1.21.11` 和 `26.1.2`，并同步现有功能、操作和界面效果。 |
| 原版信标光束 | 所有受支持版本的投影原点标记统一使用 Minecraft 原版信标光束纹理、几何和动画；保持红色 OMMC 风格并可穿过方块显示。 |
| 动态木材识别 | “任意原木”等木材选择组会从当前注册表和标签动态读取候选物品，可识别新版本及其他模组新增的木材。 |
| 界面显示扩展 | 候选悬浮面板支持配置最大行数；禁用条目统一使用保留原模型的中性灰遮罩，减少越界、闪烁和首次显示开销。 |

## 🛠 修复与体验优化

| 项目 | 说明 |
| --- | --- |
| 大型整合包性能 | 最小子材料改为分段计算与后台预热，并加入计算预算、配方查询缓存、结果签名和精确缓存失效，降低首次进入和大量材料计算时的卡顿。 |
| 整合包配方兼容 | 修复 JEI 尚未就绪时缓存错误结果、单个异常 JEI 配方清空正常配方，以及工具槽被串入递归链后产生假循环的问题；深层循环检测仅追踪单一材料转换，原木类资源保持为终端材料。 |
| 材料计算正确性 | 修复多产出配方、重复槽位、合成次数和需求数量缩放错误，以及煤炭块、白色床、骨粉/骨块、木材选择组等材料的错误拆分。 |
| JEI 交互 | 修复配方 Tooltip、配方/用途跳转、快速合成、加工台入口和“移动到容器中”按钮的兼容、状态与显示问题。 |
| 原点标记 | 修复光束消失、被实体方块、Litematica 蓝图、云层或天气遮挡的问题；保留靶心、红框黑底坐标标签和注视显示条件。 |
| 热键与 HUD | 修复热键重置按钮不能立即刷新，以及 HUD 已开启但面板不显示的问题；LMLP 不再干扰 Litematica、Tweakeroo 等其他 MaLiLib 模组的热键页面。 |
| 列表与面板 | 修复页面无法切换、列表超行、箭头和按钮异常、Tooltip 层级、面板重叠、空行及摘要定位等问题。 |
| 高版本导出按钮 | 在 `1.21.10`、`1.21.11` 和 `26.1.2` 中隐藏 Litematica 新增且与 LMLP 重叠的“原材料”和“导出”控件，保留普通文件导出与 LMLP XLSX 子材料导出。 |

## 📦 安装

1. 下载 Assets 中与 Minecraft 版本完全对应的 `litematica-material-list-plus`。
2. 删除 `mods` 文件夹中的旧版 LMLP，确保只保留一个 LMLP JAR。
3. 安装对应版本的 Fabric API、Litematica、MaLiLib 和 JEI。
