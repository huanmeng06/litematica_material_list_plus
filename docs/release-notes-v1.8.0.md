## ✨ 新功能

| 功能 | 说明 |
| --- | --- |
| 新版本支持 | 新增 Minecraft `1.21.10`、`1.21.11` 和 `26.1.2` 的 Fabric 版本，并同步现有功能、操作方式和界面效果。 |
| 原版信标光束 | 所有受支持版本的投影原点标记统一改用 Minecraft 原版信标光束纹理、几何和动画，保持红色 OMMC 风格，可穿过方块显示。 |
| 动态木材识别 | “任意原木”等木材选择组会从当前注册表和标签动态读取候选物品，新版本及其他模组新增的木材可以正常进入悬浮候选面板。 |
| 统一禁用图标 | 禁用列表条目的物品图标统一为保留原模型并覆盖中性灰遮罩的效果，避免离屏灰度转换造成的闪烁和首次显示开销。 |

## 🛠 修复与体验优化

| 项目 | 说明 |
| --- | --- |
| 大型整合包性能 | 最小子材料改为分段计算和后台预热，加入计算预算、配方查询缓存、结果签名与缓存失效机制，降低首次进入和大量材料计算时的卡顿。 |
| 循环配方 | 循环配方会提前终止，避免递归放大、重复计算和大型配方环境中的卡死。 |
| 材料数量计算 | 修复多产出配方、重复槽位、合成次数和材料需求数量的缩放错误，确保优化前后的最终展示结果一致。 |
| 材料拆分 | 修复煤炭块、白色床、骨粉/骨块和部分可逆压缩材料被错误继续拆分，以及木材选择组过度递归的问题。 |
| 原点标记层级 | 修复光束消失、被实体方块或 Litematica 蓝图遮挡的问题；光束现在位于蓝图、云层、天气和粒子之上，并保留靶心、坐标标签和注视显示条件。 |
| 热键重置 | 修复绑定热键后重置按钮仍为灰色、必须先点击搜索框才会刷新的问题，恢复与 Litematica/MaLiLib 原版一致的重置行为。 |
| JEI 交互 | 修复配方 Tooltip、配方跳转、快速合成和“移动到容器中”按钮的兼容与显示问题。 |
| 列表与面板 | 修复列表超行渲染、面板无法切换、箭头异常、Tooltip 层级和部分按钮显示错误。 |

## ⚠️ 版本支持调整

本次 Release 提供以下 Minecraft Fabric 版本：

- `1.20.1`
- `1.20.4`
- `1.20.6`
- `1.21.1`
- `1.21.10`
- `1.21.11`
- `26.1.2`

> **Minecraft 1.21.4 和 1.21.5 从 v1.8.0 起不再支持，也不会提供对应版本的 JAR。**
>
> 这两个 Minecraft 版本的最后可用版本为 LMLP `v1.7.0`，请勿将其他 Minecraft 版本的 `v1.8.0` JAR 混用到 1.21.4 或 1.21.5。

## 📦 安装

1. 下载 Assets 中与您 Minecraft 版本完全对应的 `litematica-material-list-plus`。
2. 删除 `mods` 文件夹中的旧版 LMLP，确保只保留一个 LMLP JAR。
3. 安装对应版本的 Fabric API、Litematica、MaLiLib 和 JEI。
4. LMLP v1.8.0 使用 JEI 集成，不再使用旧版 REI 集成。
