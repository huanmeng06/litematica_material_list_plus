# LMLP v1.8.1

v1.8.1 是 v1.8.0 的兼容性与交互修复版本，统一了所有受支持 Minecraft 版本的 JEI 配方体验。

## 支持的 Minecraft 版本

- `1.20.1`
- `1.20.4`
- `1.20.6`
- `1.21.1`
- `1.21.10`
- `1.21.11`
- `26.1.2`

`1.21.4` 和 `1.21.5` 仍不受支持；请继续使用其最后兼容版本 `v1.7.0`。

## 修复与改进

- 统一 LMLP 物品图标的悬浮提示为 JEI 风格，保留物品名称、模组来源及高级提示中的物品 ID；轮播材料的提示会与当前图标同步。
- 修复嵌套配方中木板、原木等可替代材料的图标轮播不同步问题，同一配方树会显示匹配的一组材质。
- 修复缺材料红色高亮的绘制范围；配方面板被滚动或遮挡时，高亮不会溢出面板。
- 修复配方转移按钮的行为一致性：普通点击只转移完成缺少合成次数所需的材料，`Shift` 点击尽可能转移全部可用材料。
- 统一配方转移按钮的 JEI Tooltip 和缺材料提示，并修复个别版本的 Tooltip 重叠、缺行与旧样式残留。

## 安装

下载与 Minecraft 版本完全一致的 JAR，删除 `mods` 文件夹内旧的 LMLP 文件，确保只保留一个 LMLP JAR。仍需安装对应版本的 Fabric API、Litematica、MaLiLib 和 JEI。
